# **HTML**

- HTML은 Tag 로 시작해 Tag 로 끝난다.
- Tag는 Box model 이다. (네모난 상자를 떠올리자.)
- Tag 는 2가지중에 하나이다 .
1) Box: header, footer, section article ..섹셔닝 혹은 그루핑 할 수 있는 것들을 말한다.
2) Item (a, button, input 등)은 사용자에게 화면에 보여지는 tag이다.
- html 파일은 항상 css 라는 스타일러 파일이 동반된다.
