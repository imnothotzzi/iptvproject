# NUMPY

**중요한거니까 알고 가야하는 것**

>[!note] Numpy 를 우리가 왜 하는가?
>  데이터 프레임이 행과 열의 형태로 나타난다. 그때 1열의 데이터를 가져 오게 되면, Series 데이터 타입 ( INDEX와 VALUE로 이루어진 타입 ) 이 나온다. 그 Series 하나 하나를 Numpy array 라고 하고, 그 array 간의 연산을 가능하게 하기 때문에 우리는 Numpy 를 배운다.
